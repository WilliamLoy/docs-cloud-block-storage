Cloud Block Storage service versions¶

The Cloud Block Storage version defines the contract and build information for the API.

Contract version¶

The contract version denotes the data model and behavior that the API supports. The requested contract version is included in all request URLs. Different contract versions of the API may be available at any given time and are not guaranteed to be compatible with one another.

Example: Example request URL (contract version is v1)

https://dfw.blockstorage.api.rackspacecloud.com/v1/1234/

Note

This document pertains to contract v1.
